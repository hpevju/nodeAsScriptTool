# nodeAsScriptTool
Demo code showing off some simple tasks that can be done with node

## Prerequisites
* [Node](https://nodejs.org/en/) 
* [git](https://gitforwindows.org/)

## To use 
* git clone 
* npm install
* modify config.json to include correct servername, port and username password
